# Week 2 Lab Activity - Student Portal Website

## Objective

The objective of this project was to create a simple 3-page website using HTML. The project helps to understand lists, tables, forms, and linking pages together.

---

## What Was Done

In this project, the following was done:

- Created 3 web pages:
  - Home page (index.html)
  - Timetable page (timetable.html)
  - Contact page (contact.html)

- Added navigation links to move between pages

- Home Page:
  - Added a heading
  - Created an ordered list (daily routine)
  - Created an unordered list (hobbies)

- Timetable Page:
  - Created a table using HTML
  - Used rowspan and colspan
  - Added a lunch break row

- Contact Page:
  - Created a simple contact form
  - Created a registration form using a table
  - Used required fields and input types like email and password

---

## Challenges Faced

- Understanding how to use tables properly
- Using rowspan and colspan correctly
- Linking pages together
