# 📱 Simple Social Media Platform (HTML Project)

## 🎯 Objective

The objective of this project was to design and develop a simple multi-page social media website using basic HTML concepts. The goal was to understand how web pages are structured, how navigation works between pages, and how to integrate multimedia and external resources into a website.

---

## 🛠️ What Was Done

In this project, the following tasks were successfully completed:

- Created a properly structured HTML document using:
  - `<!DOCTYPE html>`, `<html>`, `<head>`, `<body>`
  - Included relevant `<meta>` tags and page titles

- Developed four web pages:
  - **Home Page** (index.html)
  - **About Page** (about.html)
  - **Contact Page** (contact.html)
  - **External Links Page** (external.html)

- Implemented navigation:
  - Internal links connecting all pages
  - External links to popular social media websites

- Added content:
  - Headings and paragraphs
  - HTML comments
  - Ordered and unordered lists

- Integrated multimedia:
  - One local image
  - One online image sourced from Unsplash

---

## ⚠️ Challenges Faced

During the development of this project, several challenges were encountered:

- Managing file paths for local images and ensuring they load correctly
- Finding suitable and correctly formatted online images
- Structuring content in a clean and readable way using basic HTML only
