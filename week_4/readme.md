# Week 4 Lab Activity - CSS Positioning & Float Layout 

## Objective

The objective of this project was to practice CSS layout concepts using HTML and CSS by building a simple webpage. The focus was on positioning, floating, and the box model.

## What Was Done

* Added a fixed header
* Added a sticky navigation bar
* Created main content and sidebar sections
* Added a footer
* Used position: fixed for header
* Used position: sticky for navbar
* Used position: relative and absolute for labels
* Created two-column layout using float left and float right
* Applied box model (padding, margin, border)
* Used clear: both for footer placement

## Challenges Faced

* Positioning elements correctly using fixed, sticky, and absolute
* Understanding float layout behavior
* Managing spacing using margin and padding
* Fixing footer placement under floated elements
