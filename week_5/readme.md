# Student Grade Calculator Web Application

## Objective

To develop a simple web application using HTML, CSS, and JavaScript that calculates and displays student grades based on entered scores.

## What Was Done

* Created input fields for student name and score.
* Added Calculate Grade and Reset buttons.
* Validated user inputs.
* Calculated grades automatically using JavaScript.
* Displayed student name, score, and grade dynamically.
* Applied CSS styling and visual feedback for user interaction.
* Used DOM manipulation for dynamic updates.

## Technologies Used

* HTML5
* CSS3
* JavaScript

## Challenges Faced

* Input validation and error handling.
* Matching HTML element IDs with JavaScript code.
* Understanding DOM manipulation and event handling.
* Improving user experience through visual feedback.
