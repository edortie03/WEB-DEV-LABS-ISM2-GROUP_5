# Week 3 Lab Activity - HTML Structure & CSS Styling

## Objective

The objective of this project was to create a single webpage using HTML and CSS. The project focused on applying different CSS styling methods and using multiple selector types.

---

## What Was Done

In this project, the following was done:

- Created a single webpage containing:
  - Header section
  - Navigation menu
  - About section
  - Skills section
  - Daily routine section

- Added navigation links using HTML anchors

- Applied CSS using:
  - External CSS
  - Internal (Document Level) CSS
  - Inline CSS

- Used different CSS selectors:
  - Element selectors
  - Class selectors
  - ID selectors

- Styled:
  - Background colors
  - Typography
  - Navigation menu
  - Section headings

- Added:
  - Paragraphs
  - Lists
  - Images

---

## Challenges Faced

- Understanding the differences between Inline, Internal, and External CSS
- Managing multiple CSS selectors properly
- Styling the navigation menu
- Organizing webpage sections with proper layout and structure
